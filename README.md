# hxclient

海星商城后台管理系统

> A Vue.js project

## 安装与使用

``` bash
# 初始化项目（安装环境依赖）
npm install

# 运行开发者服务器 at localhost:8080
npm run dev

# 打包适用于生产环境的文件
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
